# React Native JSearch

## 🚀 How to use

```sh
npm start
```

## Deploy

```sh
expo publish
```

## 📝 Notes

- Get your api key from https://rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch/
- Add your API key to .env file in root
```
RAPID_API_KEY={{YOUR KEY}}
```
